Quick start

  1. Open the quickfit folder in Cursor or VS Code and open the Claude Code panel (✱ → Open in New Tab).
  2. One-time: install the Claude Code extension + CLI, run npm install in quickfit/, and vercel login.
  3. On stage you just talk to Claude — it runs the commands. The live app is already at quick-fit-neon.vercel.app; run npm run dev locally only for the build demo in Beats 3–4.
  4. Rehearse in MOCK_LLM=1 — research, build, and evals all run offline.

The 30-minute flow (you ask, Claude runs)

1 · Research ask "run the research synthesis"  ·  2 · Hypothesis ask Claude to save the bet  ·  3 · Build paste prompts P1 + P2 into the panel  ·  4 · Safe paste P3, then ask "run the eval"  ·  5 · Ship ask "deploy to Vercel" → QR on /present, the room builds, the wall fills.

Minute-by-minute words + the exact prompts are in the Run Sheet.
